# Wootalk 留言機器人

---
##### 前言：
這是一個2016的專案，目前wootalk官方針對Selenium、無痕視窗、頻繁重開視窗等異常行為，加上了驗證頁面，建議可以將main.py中，setUp(time)設高一點。
##### 環境需求：
  - python 2.x
  - Selenium IDE
  - Selenium Chrome Driver
##### 功能簡介：
1. main.py ：執行腳本。設置留言間隔時間、驗證碼輸入時間
2. msg.py：留言內容
3. web.py：Selenium腳本自動開啟視窗
